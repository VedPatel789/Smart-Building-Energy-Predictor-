# Smart Building Energy Predictor

A simple ML project that predicts heating and cooling loads of buildings using architectural features.

---

## Features
- Machine learning prediction (Random Forest)
- Interactive user inputs
- Energy load estimation
- Efficiency rating system
- Plotly visualization

---

## Inputs
- Relative Compactness
- Surface Area
- Wall Area
- Roof Area
- Height
- Glazing Area

---

## Outputs
- Heating Load
- Cooling Load
- Total Energy
- Efficiency Rating (A–D)

---

## How to Run
1. Open Google Colab
2. Upload `smart_energy_predictor.ipynb`
3. Run all cells

---

## Tech Stack
- Python
- Scikit-learn
- Pandas
- Plotly
- ipywidgets
